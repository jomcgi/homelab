   kubectl create secret generic {{ $fullName }}-api-key \

     -n {{ .Release.Namespace }}

   Then set in values:
     auth:
       existingSecret: {{ $fullName }}-api-key

2. Or set the API key directly (not recommended):
     auth:
       apiKey: sk-ant-...

{{- else if eq .Values.auth.method "manual" }}
Authentication Method: Manual

You will need to authenticate manually after the pod starts.

To authenticate, exec into the pod and run the login command:

  kubectl exec -it -n {{ .Release.Namespace }} deployment/{{ $fullName }} -- openclaw auth login

Follow the prompts to complete authentication. Your credentials will be
stored in the pod's home directory.
