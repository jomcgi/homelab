## OpenClaw MCP Integration

To connect OpenClaw to Perplexica and SearXNG, configure MCP servers in OpenClaw's settings.

### Option A: SearXNG MCP (Direct Metasearch)

For fast, lightweight lookups without LLM synthesis:

```json
{
  "mcpServers": {
    "searxng": {
      "command": "npx",
      "args": ["-y", "mcp-searxng"],
      "env": {
        "SEARXNG_URL": "http://{{ include "perplexica.fullname" . }}-searxng.{{ .Release.Namespace }}.svc.cluster.local:{{ .Values.service.searxng.port }}"
      }
    }
  }
}
```

### Option B: Perplexica MCP (AI-Powered Search)

For research-grade queries with LLM synthesis and citations:

```json
{
  "mcpServers": {
    "perplexica": {
      "command": "uvx",
      "args": ["perplexica-mcp", "stdio"],
      "env": {
        "PERPLEXICA_BACKEND_URL": "http://{{ include "perplexica.fullname" . }}.{{ .Release.Namespace }}.svc.cluster.local:{{ .Values.service.port }}/api/search",
        "PERPLEXICA_CHAT_MODEL_PROVIDER": "openai",
        "PERPLEXICA_CHAT_MODEL_NAME": "{{ .Values.llm.model }}",
        "PERPLEXICA_EMBEDDING_MODEL_PROVIDER": "openai",
        "PERPLEXICA_EMBEDDING_MODEL_NAME": "{{ .Values.embedding.model }}"
      }
    }
  }
}
```

### Network Policy Configuration

If OpenClaw uses NetworkPolicy, add egress rules to allow connections:

```yaml
networkPolicy:
  egress:
    # Allow Perplexica access
    - namespaceSelector:
        matchLabels:
          kubernetes.io/metadata.name: {{ .Release.Namespace }}
      podSelector:
        matchLabels:
          app.kubernetes.io/name: perplexica
      ports:
        - protocol: TCP
          port: {{ .Values.service.port }}
        - protocol: TCP
          port: {{ .Values.service.searxng.port }}
```

## API Endpoints

- Search API: `POST /api/search` (Perplexica)
- SearXNG API: `GET /search?q=<query>&format=json` (SearXNG)

## Focus Modes (Perplexica API)

Available focus modes for search:
- `webSearch` - General web search
- `academicSearch` - Scholarly articles and papers
- `writingAssistant` - Writing help and suggestions
- `wolframAlphaSearch` - Computational queries
- `youtubeSearch` - Video content
- `redditSearch` - Community discussions
