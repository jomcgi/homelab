## CRDs

The following CRDs are installed by the agent-sandbox subchart.
CRDs live in the chart's crds/ directory; use --skip-crds (helm) or
spec.helm.skipCrds (ArgoCD) to skip installation when agent-sandbox.installCRDs
is false:

  - sandboxes.agents.x-k8s.io
  - sandboxclaims.extensions.agents.x-k8s.io
  - sandboxtemplates.extensions.agents.x-k8s.io
  - sandboxwarmpools.extensions.agents.x-k8s.io

{{ else -}}
CRD installation is DISABLED (agent-sandbox.installCRDs: false).
Ensure these CRDs are already present in the cluster (e.g. installed by
the cluster-critical release) before sandboxes can be created.

## Parallel deployment alongside cluster-critical

To deploy agent-platform alongside an existing cluster-critical release that
already owns the agent-sandbox CRDs and controller, set in your values file:

  agent-sandbox:
    installCRDs: false
    deployController: false

If using helm install (not ArgoCD), also pass --skip-crds:

  helm install agent-platform-dev charts/agent-platform/ \
    -f overlays/dev/agent-platform/values.yaml \
    --skip-crds

With ArgoCD, also add to the Application spec:

  spec:
    source:
      helm:
        skipCrds: true
