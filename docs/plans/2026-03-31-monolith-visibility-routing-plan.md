# Monolith Visibility Routing Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Implement the two-hostname routing model with SvelteKit SSR frontend and FastAPI internal data layer — proving the full public/private stack end-to-end.

**Architecture:** Gateway rewrites `public.jomcgi.dev/*` → `/public/*` and `private.jomcgi.dev/*` → `/private/*` into SvelteKit. SvelteKit handles all user-facing HTTP via SSR. FastAPI is an internal data layer (no K8s Service, no user access). Each service colocates frontend and backend code, with symlinks generated by `format` into the SvelteKit route tree.

**Tech Stack:** SvelteKit (adapter-node, SSR), FastAPI, SQLModel, Postgres (CloudNativePG), Gateway API (Envoy), Helm, Bazel

    fetch(`${API_BASE}/api/home`, { signal: AbortSignal.timeout(10000) }),
    fetch(`${API_BASE}/api/home/schedule/today`, {

      method: "PUT",
      headers: { "Content-Type": "application/json" },

### Task 6: Add route symlink generation to format pipeline

Add a generator script that creates symlinks from service frontend directories into the SvelteKit route tree. Wire it into the `format` multirun target.

**Files:**

- Create: `projects/monolith/generate-routes.sh`
- Modify: `bazel/tools/format/BUILD` (add to multirun)

**Step 1: Create generate-routes.sh**

```bash
#!/usr/bin/env bash
set -euo pipefail

MONOLITH_DIR="projects/monolith"
ROUTES_DIR="${MONOLITH_DIR}/frontend/src/routes"

# Clean existing symlinks in routes dir
find "${ROUTES_DIR}/private" "${ROUTES_DIR}/public" -type l -delete 2>/dev/null || true
mkdir -p "${ROUTES_DIR}/private" "${ROUTES_DIR}/public"

# Generate symlinks for each service
for service_dir in "${MONOLITH_DIR}"/*/frontend; do
  service=$(basename "$(dirname "$service_dir")")
  [ "$service" = "frontend" ] && continue  # skip the SvelteKit shell itself

  for visibility in private public; do
    if [ -d "${service_dir}/${visibility}" ]; then
      ln -sf "../../../${service}/frontend/${visibility}" "${ROUTES_DIR}/${visibility}/${service}"
    fi
  done
done

# Symlink shared components to $lib
if [ -d "${MONOLITH_DIR}/shared/frontend/components" ]; then
  ln -sf "../../shared/frontend/components" "${ROUTES_DIR}/../lib"
fi
```

**Step 2: Add to format pipeline**

Add a `sh_binary` target in the monolith BUILD and add it to the format multirun in `bazel/tools/format/BUILD`.

**Step 3: Test the script**

```bash

```

**Step 4: Commit**
